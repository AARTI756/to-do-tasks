# TaskFlow Pro

TaskFlow Pro is an advanced task management application built with React, Tailwind CSS, and Framer Motion. It offers a clean, modern, and responsive interface for organizing your daily tasks with precision.

## Features

- **Advanced Task Management**: Add, edit, and delete tasks with ease.
- **Categorization**: Organize tasks into categories like Work, Personal, Study, or Other.
- **Priority Levels**: Assign High, Medium, or Low priority to your tasks.
- **Due Dates**: Set deadlines for your tasks and track them visually.
- **Real-time Search & Filtering**: Quickly find tasks using the search bar or filter by category, priority, and status.
- **Progress Tracking**: A dynamic progress bar shows your completion percentage.
- **Dark Mode**: Seamlessly toggle between light and dark themes.
- **Local Persistence**: All your tasks and preferences are saved to `localStorage`.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Smooth Animations**: Interactive UI elements powered by `motion`.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS version recommended)

### Installation

1. **Extract the project files** to your desired location.
2. **Open the project folder** in VS Code.
3. **Open the Integrated Terminal** in VS Code.
4. **Install dependencies**:
   ```bash
   npm install
   ```

### Running the Application

1. **Start the development server**:
   ```bash
   npm run dev
   ```
2. **Open your browser** and navigate to the URL shown in the terminal (usually `http://localhost:3000` or `http://localhost:5173`).

## Technologies Used

- **React**: Frontend library for building the UI.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Lucide React**: Beautifully simple, pixel-perfect icons.
- **Framer Motion**: Animation library for smooth transitions.
- **Vite**: Next-generation frontend tooling for fast development.
